# Agriculture Equipment Rental System (AERS)

A web-based platform that connects farmers with agricultural equipment owners for renting machinery.

## Tech Stack
- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: Core PHP
- Database: MySQL
- Server: Apache (XAMPP)

## Architecture
Three-Tier Architecture:
- Presentation Layer
- Application Layer
- Data Layer

## Setup
1. Install XAMPP
2. Clone the repository
3. Place project inside `htdocs`
4. Configure `.env`
5. Start Apache & MySQL
